# protractor-sandbox

- `divshot s` to serve
- if setup, `firebase serve` to serve via firebase
- `protrator testing/conf.js` to test
- `testing/params.js` for test params
- `protractor testing/conf.js --params.port 5000` run on firebase (if you're setup)
